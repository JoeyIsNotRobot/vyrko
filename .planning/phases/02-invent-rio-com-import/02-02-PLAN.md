    Ler o arquivo inteiro. Localizar o handler `document.addEventListener('submit', async (event) => {` que processa `form[data-career-ajax]` (começa na linha ~126).

    Dentro desse handler, APÓS a linha `event.preventDefault();` e ANTES do bloco `try {`, adicionar a lógica de detecção do form de import:

    ```
    let importTimers = [];
    const isImport = form.dataset.importForm !== undefined;

    if (isImport) {
        window.dispatchEvent(new CustomEvent('open-loading-modal', {
            detail: { steps: ['Lendo documento', 'Extraindo dados', 'Organizando perfil', 'Concluído'] }
        }));
        importTimers.push(setTimeout(() => {
            window.dispatchEvent(new CustomEvent('advance-loading-modal', { detail: { step: 1 } }));
        }, 800));
        importTimers.push(setTimeout(() => {
            window.dispatchEvent(new CustomEvent('advance-loading-modal', { detail: { step: 2 } }));
        }, 1800));
    }
    ```

    Dentro do bloco `try { ... }`, após `const response = await fetch(...)` e antes do `const text = await response.text()`, adicionar:
    ```
    if (isImport) importTimers.forEach(clearTimeout);
    ```

    No bloco de sucesso (após `if (!response.ok)`), no branch de sucesso (onde está `showMessage(...)`), adicionar ANTES de showMessage:
    ```
    if (isImport) {
        window.dispatchEvent(new CustomEvent('succeed-loading-modal'));
        window.dispatchEvent(new CustomEvent('import:success'));
    }
    ```

    No bloco `if (!response.ok)` (tratamento de erro), adicionar:
    ```
    if (isImport) {
        const errMsg = payload.message || (payload.errors ? Object.values(payload.errors).flat()[0] : null) || 'Algo deu errado ao processar o arquivo. Tente novamente em alguns instantes.';
        window.dispatchEvent(new CustomEvent('fail-loading-modal', { detail: { message: errMsg } }));
        return;
    }
    ```
    Colocar ANTES do `showErrors(...)` existente para que o import use o modal de erro em vez da caixa de erros inline.

    No bloco `catch (error)` (linha ~178), adicionar:
    ```
    if (isImport) {
        window.dispatchEvent(new CustomEvent('fail-loading-modal', { detail: { message: error.message || 'Algo deu errado. Tente novamente.' } }));
        return;
    }
    ```
    ANTES do `showErrors([error.message])` existente.

    IMPORTANTE: As variáveis `importTimers` e `isImport` devem ser declaradas com `let`/`const` no escopo do handler (antes do try), não com `var` global. Isso garante que cada submit tenha seus próprios timers.

    Após as modificações, rodar `npm run build` para verificar que não há erros de sintaxe JS.

| Boundary | Description |
|----------|-------------|
| CustomEvent window → Alpine component | Eventos disparados pelo próprio app.js — sem input externo |
| Modal error message | Mensagem vem do payload JSON do servidor ou do catch de network — não de input do usuário |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-05 | Denial of Service | setTimeout acumulado sem clearTimeout | mitigate | clearTimeout(importTimers) chamado ao receber resposta — timers não se acumulam entre submits |
