    2. Em resources/js/app.js, adicionar no TOPO do arquivo (antes do código vanilla existente):

       import Alpine from 'alpinejs';
       window.Alpine = Alpine;

       Alpine.data('loadingModal', () => ({
           show: false,
           progress: 0,
           currentStep: '',
           steps: [],
           error: null,
           open(steps) {
               this.steps = steps;
               this.currentStep = steps[0] ?? '';
               this.progress = 0;
               this.error = null;
               this.show = true;
           },
           advance(stepIndex) {
               this.currentStep = this.steps[stepIndex] ?? '';
               this.progress = Math.round(((stepIndex + 1) / this.steps.length) * 100);
           },
           succeed() {
               this.progress = 100;
               setTimeout(() => { this.show = false; }, 400);
           },
           fail(message) {
               this.error = message;
           }
       }));

       Alpine.start();

    3. O código vanilla JS existente (menuToggle, data-loading-link, data-locale-select, form submits) NÃO deve ser alterado — Alpine corre independentemente via Alpine.start().

    ATENÇÃO: app.js usa `type: module` (veja package.json "type": "module") — import no topo é válido.
  </action>
